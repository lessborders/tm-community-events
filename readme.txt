== Description ==

This plugin extends the TicketMachine Event Manager & Calendar Plugin. Give your users the ability to manage and add their own events to your community calendar!

* Allow users to create events without requiring admin access
* Users can create and manage their own organizer
* Admins can accept or reject events before they are published
* Mobile friendly responsive design
* Translatable with WPML
* BiTV & WCAG compliant (Accessibility for all)
* W3C Valid

When using the TicketMachine Community Events plugin, you must have the TicketMachine Event Manager & Calendar plugin installed.

== Installation ==

### Install with wordpress

1. Click "Install" at the top of the plugin page.
2. Click "Activate".
3. Done! Users can now add their events on a new page.

### Manually install the plugin

1. Go to https://wordpress.org/plugins/ticketmachine-community-events/ 
2. Click "Download" at the top of the plugin page.
3. In the Wordpress backend go to "Plugins" and click "Install".
3. Click "Upload Plugin".
4. Select the ZIP file that you downloaded from the plugin page and press "Ok".
5. Click "Activate".
6. Done! You can now start managing your events.
